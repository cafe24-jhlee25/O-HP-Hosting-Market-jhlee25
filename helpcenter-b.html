더 알아보기: 운영 대행 기능

마켓플러스 기반으로 브랜드몰과 마켓 간 상품 정보를 자동 연동합니다.

 

기능 설명
상품 자동 연동 브랜드몰에 진열/판매중인 상품이 마켓에 자동 등록. 수정도 브랜드몰에서만 하면 자동 반영.
이미지노출 + SEO 별도 이미지 등록 없이 마켓에 노출. 마켓별 검색 최적화로 상품 노출 기회 확대.
마켓별 검색 최적화 키워드 설정, 검색필터, 상품명 개선 등 마켓별 SEO 최적화로 노출 기회 확대.
마켓별 판매가 설정 브랜드몰 판매가 기준 마켓마다 할인/할증을 다르게 설정 가능.

 

 

자주 묻는 질문

 

Q. 이용 요금은 어떻게 되나요?

브랜드몰 수수료는 기준금액의 2%, 마켓 수수료는 1%입니다. (부가세 별도)
기준금액은 실결제금액에서 배송비를 차감한 금액이며, PG사 정산 시 선취됩니다.

 

Q. 마켓플러스를 꼭 이용해야 하나요?

네, 마켓 서비스는 마켓플러스 기반으로 운영됩니다. 마켓플러스 미이용 시 이용 불가합니다.

 

Q. 기존 마켓 관리 프로그램과 병행 가능한가요?

각 마켓 판매자센터에서 API 연동업체를 '카페24'로 설정해야 합니다. 마켓 정책에 따라 병행 여부가 달라집니다.

 

Q. 모든 상품이 마켓에 등록되나요?

브랜드몰의 진열 및 판매중인 상품만 대상이며, 마켓 정책에 부합하지 않는 상품은 제외될 수 있습니다.

 

Q. 아마존 연동 시 특별한 절차가 있나요?

아마존 셀러센트럴 계정으로 OAuth 인증하면 간편하게 연동됩니다.

 

Q. 마켓 상품이 바로 판매 상태가 되나요?

"브랜드몰 상품 마켓 등록 완료" 메시지 수신 후 3영업일 이내에 연동 해제가 가능합니다. 기한 내 미조치 시 자동으로 판매 상태로 전환됩니다.
